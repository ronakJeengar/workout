# Release Notes - V1.0.0

## What's New
- **V1 Launch:** The official first release of the Workout Tracker!
- **Full Localization:** Complete support for English, Spanish (Español), and Hindi (हिन्दी).
- **Isar Engine:** High-performance database backend for lightning-fast history and program management.
- **Material 3 Design:** A modern, beautiful, and accessible user interface with support for Light and Dark modes.

## Improvements
- Optimized memory usage for long workout sessions.
- Improved data migration path for users upgrading from earlier beta versions.
- Enhanced rest timer accuracy and interactive notifications.

## Known Limitations
- Landscape mode is currently disabled; optimized for portrait orientation.
- Cloud sync not available in V1; use local backup/restore functionality for manual data transfer.
