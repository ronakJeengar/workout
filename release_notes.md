# Release Notes RC-1

**Version:** 1.0.0+1
**Status:** Release Candidate 1 (RC-1)

## Highlights
- **Localization:** Support for English, Spanish, and Hindi has been fully implemented, covering all core UI elements and dynamic content.
- **Persistence:** Upgraded data layer from SharedPreferences to Isar Database. Existing users will have their data automatically migrated without any data loss.
- **Production Branding:** Added final app icons and initial store assets.
- **Stability:** Fixed remaining UI bugs, deprecated widget usage, and achieved a 100% green build pipeline.

This candidate represents a complete feature-freeze. Testing should focus on data migration and translation accuracy.
