# GEMINI.md

Project: Workout App

## Goal

Build a modern workout tracking app with clean architecture and excellent UX.

## Tech Stack

* Flutter (stable)
* Riverpod
* GoRouter
* Material 3

## Architecture Rules

* Feature-first structure
* Separate data/domain/presentation
* Keep widgets small
* Prefer composition
* Use immutable models

## Coding Rules

* Use const whenever possible
* Avoid duplicate widgets
* Strong typing
* No unnecessary comments

## State Management

* Riverpod only

## Navigation

* GoRouter only

## Testing

* Generate widget tests for UI
* Add unit tests for business logic

## Workflow

1. Analyze before editing
2. Show planned changes
3. Edit only approved files
4. Keep files under ~300 lines
